## プレビュー方法

`index.html` をブラウザで直接開くだけでプレビュー可能です。
Google Fonts(Noto Sans JP / Montserrat)をCDNから読み込んでいるため、初回表示時にインターネット接続が必要です。

## 差し替え必要箇所

### 画像プレースホルダー
- `[ MAIN PHOTO ]` — MVメインビジュアル(代表者または候補者集合写真)
- `[ PHOTO ]`(メッセージ部) — 県連代表顔写真
- `[ PHOTO ]`(仲間たち部 × 5) — 所属議員の顔写真

画像は `assets/images/` に配置し、該当箇所を `<img>` タグに差し替えてください。

### テキスト
- 代表者名: `代表 〇〇 〇〇`
- 議員名 A-E(仲間たちセクション)
- 重点公募地域リスト(現状はサンプル、実際の地域に差し替え)
- 応募状況バッジ(`status-open` / `status-few` / `status-closed`)
- 活動スケジュール(月次更新)
- フッター住所・電話番号
- SNSリンクURL

### リンク先URL
- SNS各種(ヘッダー・フッター)
- オンライン応募フォーム(Googleフォーム等)
- 政治塾・サポーター登録ページ

## レスポンシブ対応

768px以下でモバイルレイアウトに切り替わります。
- MVが縦積み(テキスト上 / ビジュアル下)
- 段階的参画ファネルが縦積み
- 仲間たちが2列グリッド
- イベント詳細が1カラムに
- 応募フローが縦並びに
- 右端固定CTAは非表示

## Rev.2 設計思想

デザイン品質(国民民主党サイトの強み)と行動誘導力(維新サイトの強み)の両立を狙う。

**応募までのファネル**:
```
MV(ブランド認知)
  ↓
段階的参画ファネル(関わり方の選択肢を提示)
  ↓
5/24イベント告知(具体的な次の行動)
  ↓
メッセージ・仲間たち(心理的共感・同質性)
  ↓
募集要項・フロー(STEP 00で心理的ハードル緩和)
  ↓
応募方法(2経路で行動コスト最小化)
  ↓
活動スケジュール(継続的な関与機会)
```

## 制作メモ

- カード・影・ボーダーを排除し、余白と背景色のみで要素を分離
- 1色が画面の60%以上を占める「支配色」原則を実装
- リストマーカーはすべて赤丸(●)で統一
- 見出しは極太900ウェイト、本文との太さ落差を3倍以上確保
- STEP 00のみオレンジ背景で他ステップと差別化し、視線を誘導
